Introduction to HTML

HTML (Hyper Text Markup Language) is the standard language for creating web pages.

  • Hyper Text: Text that contains links to other pages or resources.
  • Markup: Special tags (< >) that describe the structure and meaning of content.

A Simple HTML Page:

Highlighted HTML Code

        
          <!DOCTYPE html>
          <html>
            <head>
              <title>My First Page</title>
            </head>
            <body>
              <h1>Hello, World!</h1>
              <p>This is my first HTML page.</p>
            </body>
          </html>

Why HTML?

  • Define structure for all web content (text, images, videos, forms).
  • Works in every browser.
  • Forms the foundation before adding CSS (styling) or JavaScript (interactivity).

Day: 1 Basic Structure, Headings, Paragraphs, Lists

1. Structure Tags

  • <html>, <head>, <body>: Skeleton of every page.
  • Use: Always - every HTML document starts with these.

Document Structure

  • <DOCTYPE html>: Browser ko HTML5 version batane ke liye.
  • <html>: Root container.
  • <head>: Meta info - title, SEO, CSS/JS links.
  • <body>: Screen pe jo dikhega.
  • Use: Har HTML page me shuruat me.

2. Heading <h1>-<h6>

  • Use: Title & sub-titles; <h1> for the main heading, <h2> for sections, etc.

Headings <h1> - <h6>

  • h1 sabse bada, h6 sabse chhota.
  • Kab use Karain:
  • h1: Page ka main title.
  • h2: Section headings (About, Contact).
  • Niche ke headings subsections ke liye.

3. Paragrpah <p>

  • Text blocks likhne ke liye.
  • Use: regurlar text blocks, articles, blog content, descriptions.

4. Lists

  • Unordered <ul>: Bullet list (featrues, points).
  • Ordered <ol>: Numbered list (steps, rankings).
  • List Item <li> Har item ke liye.
  • Use: Navigation menus, steps features list, etc.

Day 2: Links, Images, Tables

1. Links <a>

  • Syntax <a href="page.html">Click/a>
  • Attributes: <target="_blank"> nayi tab me kholne ke liye.
  • href attribute sets destination.
  • Use: Navigation menu, download links, external resources, same-page anchor, internal page jumps.

2. Images <img>

  • Synta: <img src="image.jpg" alt="Description">
  • alt SEO + accessibility ke liye
  • srcfor path, alt for description.
  • Use: Logos, illustrations, product picutres, banners.

3. Tables <table> with <tr>, <td>, <th>

  • Use: Display tabular data (pricing, schedules, comparision).
  • Structure:

<table border="1" cellpadding="8">
  <tr>
    <th>Heading</th>
    <th>Heading</th>
  </tr>
  <tr>
    <td>Data</td>
    <td>Data</td>
  </tr>
</table>
  • Use: Timetables, price charts, comparison data.

Day 3: Forms

Collect User Input.

  • <formt>: Poora form container.
  • <input>: Text, email, password, checkbox, radio.
  • <textarea>: Multi-line text
  • <label>:Input ke naam ka tag.
  • <buttion> / <input type="submit">: Form bhejne ke liye.
  • Use Examples:
  • Contact forms, sign-up/login, feedback surveys.

Day 4: Semantic Tags

Describe meaning, not just layout. Improves SEO & accessibility. OR Semantic tags browser aur SEO ko content ka maqsad batate hain.

  • <header>: Top section (logo, nav)
  • <nav>: Main menu/links.
  • <section>: Thematic grouping of content.
  • <article>: Independent content (blog post).
  • <footer>: Bottom info(copyright, social links).
  • Use: Any modern, well-structure page. OR Modern, SEO-friendly layout banate waqt. 

| Tag         | Purpose                   | Use Example                     |
| ----------- | ------------------------- | ------------------------------- |
| <header>  | Page/section ka top area  | Logo, navigation menu           |
| <nav>     | Links ka group            | Main site menu                  |
| <section> | Theme-wise content        | Services section, About section |
| <article> | Independent content block | Blog post, news article         |
| <footer>  | Page ka bottom            | Copyright, contact links        |
