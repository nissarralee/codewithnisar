Introduction to CSS

CSS (Cascading Style Sheets) HTML page ko design aur layout dene ke liye hoti hai.

  • Cascading: Agar ek elment pe multiple styles apply ho to priority ke rules follow hote hain (inline > internal > external).
  • Purpose: Colors, spacing, fonts, animations, responsive layouts.

CSS ko page se Link Karne ke 3 Tareeqe hai.

1. External (Recomended):


<link rel="stylesheet" href="style.css">

2. Internal <style> tag <head>

3. Inline Element ke andar style="color:red;" (avoid for large projects).

Day: 8 Selectors, Colors, Backgrounds

1. Selectors

  • Use: Kis element ko stlye dena hai ye decide karte hain.
  • Kahan: Har CSS rule mein.
  • Types:
    • Element selector: p {color: blue};
    • Class selector: .title { font-size: 20px } specific group ko style karne ka liye.
    • ID slector: #main { background: yellow; }
    • Attribute/Child/Descendant: Advanced targeting.
    • Use: Decide kis element ko style karna hai.

2. Colors

  • Use: Text, background, borders ka color set karna.
  • Formats:
    • Name: red, blue
    • HEX:#ff0000
    • RGB:rg(255,0,0)
    • RGBA (with opacity):rgba(0,0,0,0.5)

3. Backgrounds

  • background-color, background-image, background-size, background-position.
  • Use: Hero sections, banners, textures, background color ya image set karna.
  • Kahan: Hero section, banners, buttons.

Example: 


body {
  background-color: #f5f5f5;
}
h1 {
  color: navy;
}

Day 9: Box Model (Margins, Padding, Borders)

  • Har HTML element ek box hta hai.
  • Box Model parts:
    • Content → Text ya image.
    • Padding → Content aur border ke beech space.
    • Border → Element ka border.
    • Margin → Element aur doosre elements ke beech space.
  • Use: Layout spacing, card designs, button stylling.

div {
  margin: 20px;
  padding: 15px;
  border: 2px solid black;
}

Day 10: Flexbox

Flexbox se 1-dimentsioanl (row ya column) layout easily banta hai. Or Layout banane ke liye elements ko row ya colun mein arragne karna.

  • Container properties: display:flex, justify-content, align-items, flex-direction.
  • Item propertites: flex, align-self
  • Use: Navbars, card layouts, horizontally/vertically centered content.

Example:


.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

Day 11: CSS Grid

Grid se 2-dimentsional layout design hota hai. Or Complex layouts banane ke liye.

  • display:grid, grid-template-columns/row, gap.
  • Combine with media quries for reponsive grids.
  • Use:Web ka full layout (header, sidebar, main, footer, galleris, dashboards.

Example:


.grid {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  gap: 10px;
}

Day 12: Transition & Animations

  • Transitions: Smooth property change (hover effects).

button {
  background: blue;
  transition: background 0.3s;
}

button:hover {
  background: red;
}
  • Animations: Keyframes definem movement or style changes.

@keyframes fade {
  from { opacity: 0; }
  to { opacity: 1; }
}
  • Use:Hover effects, loaders, banners.

Day 13: Responsive Design (Media Queries)

Responsive design = har screen size pe acha layout.

  • Different Screen sizes ke liye layout adjust karna.
  • Uses: Mobile-friendly sites, tablets, different devices.
  • Kahan: Mobile, tablet, desktop layouts.

Example:


@media (max-width: 768px) {
  body {
    font-size: 14px;
    background: lightblue;
  }
}
