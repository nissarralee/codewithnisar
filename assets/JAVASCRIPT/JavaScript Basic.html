Day 15 → Variables, Data Types, Operators

1. Variables – Data store karne ke containers.

  • var (old), let (block scoped), const (fixed value).
  • Use: User name, scores, counters.

2. Data Types

  • Primitive: String, Number, Boolean, Null, Undefined, Symbol, BigInt.
  • Non-primitive: Object, Array.
  • Use: Har tarah ka data hold karne ke liye.

3. Operators

  • Arithmetic: + - * / %
  • Comparison: ==, ===, >, <
  • Logical: &&, ||, !
  • Use: Calculations, condition checking.

Variables Example:

let userName = "Ali";
const birthYear = 2000;
var score = 95;

// Display in console
console.log(userName);

Day 16 → Functions, Conditions, Loops

1. Functions – Reusable code blocks.

function greet(name) { 
    return "Hello " + name; 
}
  • Use: Repeated tasks (calculations, DOM updates).

2. Conditions – Decision making.

  • if, else if, else, switch.
  • Use: Login checks, feature toggles.

3. Loops – Repeat actions.

  • for, while, do…while, for…of, for…in.
  • Use: Iterating arrays, repeated DOM updates.

Function & Condition Example:

function checkAge(age) {
    if (age >= 18) {
        return "Adult";
    } else {
        return "Minor";
    }
}

// Test the function
let result = checkAge(20);

Day 17 → Arrays & Objects

1. Arrays – Ordered lists of data.

  • Methods: push, pop, map, filter.
  • Use: Store items (tasks, product lists).

2. Objects – Key–value pairs.

const user = {
    name: "Ali",
    age: 20,
    city: "Mumbai"
};
  • Use: Represent entities like users, products.

Array & Object Example:

// Array example
let fruits = ["Apple", "Banana", "Orange"];
fruits.push("Mango");

// Object example
let person = {
    firstName: "John",
    lastName: "Doe",
    age: 30
};

Day 18 → DOM Manipulation

The DOM (Document Object Model) lets JS interact with HTML.

Selectors

  • document.getElementById('id')
  • document.querySelector('.class')

Change Content/Style

element.textContent = "New text";
element.style.color = "red";

Events

  • addEventListener('click', handler)
  • Use: Button clicks, form submissions, mouse moves.

Day 19 → Forms Handling & localStorage

1. Forms Handling

  • Access values: document.forms["myForm"]["username"].value
  • Validate input: Empty fields, email format.

2. localStorage

  • Save key/value pairs in browser.
  • localStorage.setItem("key","value")
  • Persist data even after page refresh (useful for dark mode, cart items).

Day 20 → ES6+ Features

Modern JS updates:

  • let & const – Better variable scoping.
  • Arrow Functions () => {} – Short syntax.
  • Template Strings `Hello ${name}` – Easy string + variables.
  • Destructuring, Spread/Rest operators – Cleaner code.
  • Use: Almost all modern JS projects.

ES6+ Features Example:

// Arrow function
const multiply = (a, b) => a * b;

// Template string
let name = "Alice";
let message = `Hello, ${name}! Welcome to JavaScript.`;

// Destructuring
let user = { firstName: "John", lastName: "Doe" };
let { firstName, lastName } = user;
