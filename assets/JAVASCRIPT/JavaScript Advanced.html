Day 22 ‚Üí Callbacks & Promises

1. Callbacks

  • Ek function ko doosre function ke argument ke taur pe dena.
  • Problem: Nested callbacks ‚Üí "callback hell".

Callback Example:

function getData(cb) { 
    setTimeout(() => {
        cb("Data received!");
    }, 1000);
}

// Using the callback
getData(result => {
    console.log(result);
});

2. Promises

  • Async operations ko handle karne ka clean tareeqa.

Promise Example:

function fetchData() {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            const success = true;
            if (success) {
                resolve("Promise resolved successfully!");
            } else {
                reject("Promise rejected!");
            }
        }, 1500);
    });
}

// Using the promise
fetchData()
    .then(result => console.log(result))
    .catch(error => console.error(error));

Day 23 ‚Üí Async/Await & Fetch API

1. Async/Await

  • Promises ko synchronous style me likhne ka tareeqa.

Async/Await Example:

async function loadData(){
    try {
        const response = await fetch('https://jsonplaceholder.typicode.com/posts/1');
        const data = await response.json();
        return data;
    } catch(error) { 
        console.error('Error:', error); 
    }
}

Day 24 ‚Üí JSON Basics

  • JSON (JavaScript Object Notation): Lightweight data format (key‚Äìvalue).
  • Parse string to object: JSON.parse(jsonString).
  • Stringify object to string: JSON.stringify(obj).
  • Use: APIs data exchange, configuration files.

JSON Example:

// JavaScript Object
const person = {
    name: "Alice",
    age: 30,
    city: "Mumbai"
};

// Convert to JSON string
const jsonString = JSON.stringify(person);
console.log(jsonString);

// Parse back to object
const parsedObject = JSON.parse(jsonString);
console.log(parsedObject.name);

Day 26 ‚Üí Modules (import/export)

  • Export code from one file, Import in another.
  • export / export default and import syntax.
  • Use: Large projects ko reusable chhote files me todne ke liye.

Module Example:

// math.js (exporting module)
export const pi = 3.14159;

export function add(a, b) {
    return a + b;
}

export default function multiply(a, b) {
    return a * b;
}

// app.js (importing module)
import multiply, { pi, add } from './math.js';

console.log(`PI value: ${pi}`);
console.log(add(5, 3));
console.log(multiply(4, 7));

Note: Modules require separate files and a server environment to work properly.

Day 29 ‚Üí Git & GitHub Intro

2. Remote on GitHub

  • git push -u origin main

# Commit changes with a message
