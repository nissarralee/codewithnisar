CSS Grid Complete Tutorial

Two-dimensional layout system for modern web design

1. CSS Grid Kya Hai?

  • Two-Dimensional: Rows aur columns ko ek sath control karta hai.
  • Responsive: Different screen sizes pe layout adjust karna simple.
  • Precise Control: Har item ki exact jagah define kar sakte hain.

Flexbox sirf ek dimension (row ya column) handle karta hai, lekin Grid ke sath aap complex web layouts easily design kar sakte hain.

2. Basic Setup

Grid banane ke liye parent element par:

.container {
  display: grid;
}

Container ke andar jitne direct children hain wo grid items ban jate hain.

3. Important Concepts

  • Grid Container: Parent element with display: grid
  • Grid Items: Direct child elements
  • Grid Lines: Horizontal (rows) aur vertical (columns) imaginary lines
  • Grid Tracks: Rows aur columns ka area
  • Grid Cells: Row + column intersection (single box)
  • Grid Area: Multiple cells ka combined area

6. Named Grid Areas

Layout ko naam dekar manage karna:

.container {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
  grid-template-columns: 200px 1fr;
  grid-template-rows: 60px 1fr 50px;
}

.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }

Useful Properties Quick List

Property Description
display: grid / inline-grid Grid container
grid-template-rows / columns Define rows/columns
grid-template-areas Named layout
grid-column / grid-row Item start/end
gap, row-gap, column-gap Space between cells
justify-items / align-items Item alignment
justify-content / align-content Whole grid alignment

Tips

  • Use `fr` unit for flexible space
  • `minmax(min, max)` for responsive columns
  • Combine CSS Grid + media queries for advanced responsive layouts
  • Complex 2D layouts (dashboard, portfolio, magazines) ‚Üí Grid is best
