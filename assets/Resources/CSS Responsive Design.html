2Ô∏è‚É£ Core Principles

1. Fluid Layouts (Flexible Grid)

Fixed pixel widths ke bajay `%` ya `fr` units use karein

.container {
  width: 90%;   /* screen width ke hisaab se adjust */
  margin: auto;
}

2. Responsive Images

Images ko parent width ke hisaab se adjust karna:

img {
  max-width: 100%;
  height: auto;
}

3. Media Queries

Screen size ke hisaab se different CSS rules apply:

@media (max-width: 768px) {
  body {
    font-size: 16px;
  }
}

üëâ Yahan 768px se chhoti screen par font size change hoga

4. Mobile-First Approach

Pehle mobile ke liye CSS likho, phir bade screens ke liye min-width media queries use karo

3Ô∏è‚É£ Meta Viewport Tag (Zaroori!)

HTML `` me ye tag responsive scaling ke liye:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Without this tag, mobile devices pe website zoomed out dikhegi!
