CSS Flexbox Complete Guide

Yeh guide aapko CSS Flexbox ko step-by-step detail aur interactive examples ke sath samjhayegi

1. Flexbox Kya Hai?

Flexbox (Flexible Box Layout) CSS ka one-dimensional layout system hai. Iska kaam elements ko row (line) ya column (vertically) arrange karna aur unka size/space automatically adjust karna hai.

Kyun use karein?

  • Responsive Design: Mobile se desktop tak layout adjust hota hai.
  • Easy Centering: Element ko horizontally + vertically center karna bohat simple.
  • Less Code: Float ya positioning ki tension nahi.

2. Flex Container Shuru Karna

Flexbox activate karne ke liye parent element par:

.container {
  display: flex;      /* block level flex */
  /* ya */
  display: inline-flex; /* inline flex */
}

Container ke andar jitne bhi direct child elements hain, wo flex items ban jate hain.

3. Main Axis vs Cross Axis

Flexbox ko aik road ki tarah socho:

  • Main Axis: Jis direction main items flow karte hain.
  • Cross Axis: Main axis ke perpendicular direction.

7. Tips & Best Practices

  • `gap` use karein margin ki jagah par clean spacing ke liye.
  • Equal width ke liye `flex: 1;` best hai.
  • Responsive design ke liye media queries sath use karein.
  • 2D grid layout ke liye CSS Grid consider karein.

Quick Cheat Sheet

/* Container Properties */
.container {
  display: flex;
  flex-direction: row | column;
  flex-wrap: wrap | nowrap;
  justify-content: center | space-between;
  align-items: center | stretch;
  align-content: space-around;
  gap: 20px;
}

/* Item Properties */
.item {
  order: 0;
  flex: 1 1 auto;
  align-self: flex-start;
}
